SMS Everdrive source
====================

This project is a fork of the project found at https://github.com/smspower/sms-everdrive-os

This firmware works with the Chinese Everdrive sold on AliExpress.

------------------------------------------------------------------------------------
To configure the development environment:
------------------------------------------------------------------------------------

1) Install SDCC 3.3 in the c:\SDCC-3.3 folder

- Download SDCC from https://sourceforge.net/projects/sdcc/files/sdcc-win64/3.3.0/

2) Open the "Source" folder in Visual Studio Code in order to modify the source code

------------------------------------------------------------------------------------
To compile the project:
------------------------------------------------------------------------------------

1) Run the "_build.bat" file in the Visual Studio Code terminal

2) Copy the MSOS.mso file to the Everdrive (this is the firmware used in the Everdrive)

------------------------------------------------------------------------------------
Notes:
------------------------------------------------------------------------------------

- ​​The tools in the "everdrive-sdk" folder were obtained from the "mega-ed-sdk.7z" file, which is available at https://krikzz.com/pub/support/mega-everdrive/v1/dev/

- The "_clean.bat" file can be used to delete the files generated during compilation.
